# Guraduation_Resarch
##メインのソースコードはすべてsrc階層にあります。
```
src/
  ├ attach_img2mp4.cpp         //2枚の画像をつないで動画にするプログラム
  ├ binaly.cpp                 //入力したバイナリ文字列をASCIIに変換して表示するプログラム
  ├ canny.cpp                  //1からCannyのエッジ検出を実装しようとしたプログラム 
  ├ dvide.cpp                  //入力した画像を20x20の分割画像にして出力するプログラム
  ├ exif.cpp                   //exifライブラリを使って写真のメタデータを読み取るプログラム
  ├ glid.cpp                   //入力した画像にグリッド線を引くプログラム
  ├ hist.cpp                   //入力した画像の勾配強度を出力するプログラム
  ├ img2mp4.cpp                //入力した画像群を動画にするプログラム
  ├ match.cpp                  //自己位置推定の初期プログラム
  ├ match_Tsukuba_Canny.cpp    //つくばチャレンジで得られたデータでエッジ特徴を用いた自己位置推定を行うプログラム
  ├ match_Tsukuba_HOG.cpp      //つくばチャレンジで得られたデータでHOG特徴を用いた自己位置推定を行うプログラム
  ├ match_Tsukuba_LBP.cpp      //つくばチャレンジで得られたデータでLBP特徴を用いた自己位置推定を行うプログラム
  ├ mov2img.cpp                //動画から画像群をつくるプログラム
  ├ proto_binaly.cpp           //バイナリ文字列からASCII文字に変換する練習プログラム
  ├ small.cpp                  //画像を縮小するプログラム
  ├ time.cpp                   //HOG特徴とLBP特徴の実行時間を比較するプログラム
```
